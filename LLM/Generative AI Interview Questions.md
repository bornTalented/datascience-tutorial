# 🧠 Generative AI Interview Questions (Beginner → Advanced)

---

## 1. What is Generative AI?

**Answer:**
Generative AI (Gen AI) refers to AI systems that can **create new content**—such as text, images, audio, or code—by learning patterns from existing data.

Unlike traditional AI, which focuses on prediction or classification, Gen AI **generates original outputs**.
For example:

* Chatbots can write human-like text
* Image models can create visuals from prompts

It typically uses techniques like **transformers, deep learning, and diffusion models**.

---

## 2. How is Generative AI different from Traditional AI?

**Answer:**

| Traditional AI                       | Generative AI                 |
| ------------------------------------ | ----------------------------- |
| Focuses on prediction/classification | Focuses on content creation   |
| Example: Spam detection              | Example: Writing emails       |
| Uses models like SVM, decision trees | Uses transformers (e.g., GPT) |

**Key idea:**
Traditional AI answers *“What is this?”*
Gen AI answers *“Create something new.”*

---

## 3. What is a Large Language Model (LLM)?

**Answer:**
A Large Language Model (LLM) is a type of Gen AI trained on massive text data to **understand and generate human-like language**.

* Works by predicting the **next word in a sequence**
* Uses **transformer architecture & self-attention**
* Can perform tasks like:

  * Summarization
  * Chatbots
  * Code generation

---

## 4. What are common use cases of Generative AI?

**Answer:**
Gen AI is widely used across industries:

* **Marketing:** Ad copy, content generation
* **Healthcare:** Drug discovery
* **Software:** Code generation
* **Education:** Personalized learning
* **Media:** Script writing, voice synthesis

---

## 5. How would you automate customer support using Gen AI?

**Answer:**

1. Identify common queries (FAQs, tickets)
2. Use/fine-tune an LLM
3. Build chatbot for routine tasks
4. Add **human-in-the-loop** for complex queries
5. Apply guardrails (filters, prompt control)
6. Track KPIs:

   * Response time
   * Customer satisfaction
   * Cost reduction

## 11. What is Prompt Engineering?

**Answer:**
Designing effective prompts to guide AI output.

👉 Example:
“Summarize in 5 bullet points for executives” is better than “Summarize this.”

---

## 12. What is Fine-tuning?

**Answer:**
Training a pre-trained model on **domain-specific data** to improve accuracy.

👉 Example: Medical chatbot trained on healthcare data.

## 15. What is RAG (Retrieval-Augmented Generation)?

**Answer:**
RAG combines:

* LLM + external knowledge base

It retrieves relevant data before generating answers → improves **accuracy and reduces hallucinations**.

## 17. What are diffusion models?

**Answer:**
Models that generate images by:

* Starting with noise
* Gradually refining into an image

👉 Used in tools like image generators
